Wago is a cloud-hosted WhatsApp API that lets you send messages, receive real-time webhooks, and manage WhatsApp connections — without running any infrastructure yourself.

## Quick example

<Tabs items={["TypeScript", "Python", "cURL"]}>
  <Tab value="TypeScript">
```typescript
import { Wago } from '@wago/sdk';

const client = new Wago({ apiKey: 'wh_...' });

// Send a message
await client.sendMessage(connectionId, '1234567890@s.whatsapp.net', 'Hello!');

// Set up a webhook for incoming messages
await client.createWebhook(connectionId, 'https://your-server.com/webhook');
```
  </Tab>
  <Tab value="Python">
```python
from wago import Wago

client = Wago(api_key="wh_...")

# Send a message
client.send_message(connection_id, chat_id="1234567890@s.whatsapp.net", text="Hello!")

# Set up a webhook for incoming messages
client.create_webhook(connection_id, url="https://your-server.com/webhook")
```
  </Tab>
  <Tab value="cURL">
```bash
# Send a message
curl -X POST https://api.wago.com/api/connections/CONNECTION_ID/send \
  -H "Authorization: Bearer wh_..." \
  -H "Content-Type: application/json" \
  -d '{"chatId": "1234567890@s.whatsapp.net", "text": "Hello!"}'

# Create a webhook
curl -X POST https://api.wago.com/api/connections/CONNECTION_ID/webhooks \
  -H "Authorization: Bearer wh_..." \
  -H "Content-Type: application/json" \
  -d '{"url": "https://your-server.com/webhook", "events": ["*"]}'
```
  </Tab>
</Tabs>

## How it works

<Steps>
  <Step>
    ### Connect a WhatsApp number
    Create a connection and scan a QR code with WhatsApp to link your account. Wago provisions a dedicated cloud session that stays connected 24/7.
  </Step>
  <Step>
    ### Configure webhooks
    Point webhook URLs at your server. When messages arrive, Wago delivers them as signed HTTP POST requests with automatic retries.
  </Step>
  <Step>
    ### Build your app
    Use the TypeScript or Python SDK to send messages, fetch chats, and manage connections. Or call the REST API directly.
  </Step>
</Steps>

## Why Wago

<Cards>
  <Card title="Zero infrastructure" href="/docs/concepts/connections">
    Wago runs WAHA sessions in Kubernetes with automatic health monitoring, failover, and scaling. You just call the API.
  </Card>
  <Card title="Reliable webhooks" href="/docs/concepts/webhooks">
    HMAC-SHA256 signed deliveries with exponential backoff (5 retries), delivery logs, and a dead-letter queue for failed events.
  </Card>
  <Card title="Official SDKs" href="/docs/sdk/typescript">
    TypeScript and Python SDKs with full API coverage. Install with `npm install @wago/sdk` or `pip install wago`.
  </Card>
  <Card title="CLI & API tokens" href="/docs/cli/installation">
    Manage everything from the terminal. Browser-based login, auto-refreshing tokens, and long-lived API keys for your apps.
  </Card>
  <Card title="MCP Server" href="/docs/mcp">
    Use Wago from Claude, ChatGPT, Cursor, and other AI assistants via the Model Context Protocol.
  </Card>
</Cards>
